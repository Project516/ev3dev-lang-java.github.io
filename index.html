Why use this project?

Introduction

EV3-lang-java is a Java project to build software for robots inheriting concepts from LeJOS.

The library interact with EV3Dev, a Debian Linux-based operating system which expose a interface to interact with Sensors & Actuators in an easy way.

Why use this project?

What is the innovation?

  • Java development on top of a complete Linux distro
  • Possibility to interact with others programming languages
  • Usage of the same library for EV3 Brick, BrickPi+ & PiStorms
  • Support for Raspberry Pi
  • Modular design

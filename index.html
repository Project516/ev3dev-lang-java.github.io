Why use this project?

Introduction

EV3-lang-java is a Java project to build software for robots inheriting concepts of LeJOS and interacting with EV3Dev, a Debian Linux-based operating which expose a interface to interact with Sensors & Actuators in a easy way.

Why use this project?

What is the innovation in this library?

  • Java development on a complete Linux distro
  • Possibility to interact with others programming languages
  • EV3Dev interface
  • Support for Raspberry Pi
  • Modular design
  • Maven/Graddle support
  • Logging support

Why use this project?

The project is divided in some libraries. At the moment, it was released a Core library to interact with the hardware managed by EV3Dev.

Throughout 2017, will be developed a new library to cover the area about Mobile Robot Navigation and another about Behaviour based robotics.

For other areas, you will use other libraries, you don´t have any restriction in System side and Development side.

Everyone is welcome to add new fresh ideas to Java for Lego Mindstorms.

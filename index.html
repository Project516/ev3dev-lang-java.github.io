EV3Dev-lang-Java

A Java library designed to interact with the hardware managed by EV3Dev using the LeJOS way.

Github project |  Create a issue |  Javadoc

Why use this library?

EV3-lang-java is a library to manage Lego Mindstorms hardware using LeJOS concepts over EV3Dev, a Debian Linux-based operating.

Why use this library?

What is the innovation in this library?

  • Java library running on EV3Dev
  • Support for Raspberry Pi
  • Modular design

Why use this library?

This library put the focus in the interaction with the hardware so it doesn't cover areas as: Mobile Robot Navigation, Computer Vision, Behaviour based robotics, etc...

For other areas, you will use other libraries. Following a modular architecture to increase the flexibility.

Features (v0.2.0)

  • Regulated Motor Support
  • Unregulated Motor Support
  • Sensor Support
  • OpenCV Computer Vision Support
  • Sounds
  • LeJOS Sensor filter Support
  • eSpeak TTS (Text to speech) Support

Getting Started

Learn to use this library step by step.

1. Install EV3Dev (1h.)

Install the operating system to use this library.

Later, make a SSH connection to upgrade your brick: 


apt-get update
apt-get upgrade
apt-get dist-upgrade

2. Install Java JDK (1h.)

Install with the package manager Apt Java JDK, OpenCV for Java and some commons Java build tools. 


sudo apt-get install default-jdk
sudo apt-get install libopencv2.4-java
sudo apt-get install maven
sudo apt-get install ant

3. Create a Hello World Maven project (2m.)

Create a folder with the name: helloworld and create a new Maven project. 


mvn archetype:generate 
-DgroupId=ev3dev.java.examples 
-DartifactId=helloworld 
-DarchetypeArtifactId=maven-archetype-quickstart 
-DinteractiveMode=false
cd helloworld
mvn eclipse:eclipse

4. Import in Eclipse IDE your Maven Project (5m.)

Import in Eclipse your new Java project.

5. Add the dependency (5m.)

Update your pom.xml file the add the repository: 



    jitpack.io
    https://jitpack.io

and the artifact: 



    com.github.jabrena
    ev3dev-lang-java
    v0.2.0

6. Code an example (5m.)

Create a new Java Class with the name "Test" and copy and paste this example: 


package ev3dev.java.examples;

import ev3dev.hardware.Battery;
import ev3dev.hardware.port.MotorPort;
import ev3dev.hardware.port.SensorPort;
import ev3dev.hardware.motor.EV3LargeRegulatedMotor;
import ev3dev.hardware.sensor.ev3.EV3IRSensor;
import lejos.robotics.SampleProvider;
import lejos.utility.Delay;

public class BumperCar {
    
    //Robot Definition
    private final static EV3LargeRegulatedMotor mA = new EV3LargeRegulatedMotor(MotorPort.A);
    private final static EV3LargeRegulatedMotor mB = new EV3LargeRegulatedMotor(MotorPort.B);
    private final static EV3IRSensor ir1 = new EV3IRSensor(SensorPort.S2);

    //Configuration
    private final static int motorSpeed = 500;
    
    public static void main(String[] args) {
        
        final SampleProvider sp = ir1.getDistanceMode();
        int distance = 255;

        final int distance_threshold = 35;
        
        //Robot control loop
        final int iteration_threshold = 100;
        for(int i = 0; i <= iteration_threshold; i++) {
            forward();

            float [] sample = new float[sp.sampleSize()];
            sp.fetchSample(sample, 0);
            distance = (int)sample[0];
            if(distance <= distance_threshold){
                backwardWithTurn();
            }

            System.out.println("Iteration: " + i);
            System.out.println("Battery: " + Battery.getVoltage());
            System.out.println("Distance: " + distance);
            System.out.println();
        }

        mA.stop();
        mB.stop();
        System.exit(0);
    }
    
    private static void forward(){
        mA.setSpeed(motorSpeed);
        mB.setSpeed(motorSpeed);
        mA.forward();
        mB.forward();
    }
    
    private static void backwardWithTurn(){
        mA.backward();
        mB.backward();
        Delay.msDelay(1000);
        mA.stop();
        mB.stop();
        mA.backward();
        mB.forward();
        Delay.msDelay(1000);
        mA.stop();
        mB.stop();
    }
}

7. Build & deploy (15m.)

Create the artifact (*.jar): 


mvn package

and deploy your .jar in your brick 


scp ev3-lang-java-0.2-SNAPSHOT.jar your_username@remotehost:/home
scp helloworld-1.0-SNAPSHOT.jar your_username@remotehost:/home

8. Run your example (2m.)

Connect with your brick: 


ssh your_username@remotehost
your_password
cd /home

and finally, run your example: 


java -cp 
    helloworld-1.0-SNAPSHOT.jar:ev3-lang-java-0.2-SNAPSHOT.jar 
    ev3dev.java.examples.Test

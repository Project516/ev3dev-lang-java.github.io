EV3Dev-lang-Java

A Java library to interact with EV3Dev hardware and build Software for educational Robots. v0.3.0

Github project |  Create a issue |  Javadoc |  Examples

                 

Why use this library?

EV3-lang-java is a Java library to manage Lego Mindstorms hardware using LeJOS concepts over EV3Dev, a Debian Linux-based operating.

Why use this library?

What is the innovation in this library?

  • Java library running on EV3Dev
  • Support for Raspberry Pi
  • Modular design
  • Logging support

Why use this library?

This library put the focus in the interaction with the hardware so it doesn't cover areas as: Mobile Robot Navigation, Computer Vision, Behaviour based robotics, etc...

For other areas, you will use other libraries. Following a modular architecture to increase the flexibility.

Features (v0.3.0)

  • Regulated Motor Support
  • Unregulated Motor Support
  • Sensor Support
  • OpenCV Computer Vision Support
  • Sounds
  • LeJOS Sensor filter Support
  • eSpeak TTS (Text to speech) Support
  • Log support
